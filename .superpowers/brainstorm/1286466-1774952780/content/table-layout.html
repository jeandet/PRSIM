Layout Structure

Node Tree:
TableNode
├── HeaderRow (fixed top)
├── Cell[0..N]
├── ScrollRegion (V+H scroll)
├── Row[i] (recycled pool)
├── Cell[0..N]
└── HScrollbar (if overflow)
Key Properties:
• Uniform row height
• Equal column widths (v1)
• Header scrolls with body (H)
• Header stays fixed (V)
• Row recycling from pool
• Selected row = highlight fill
• Alternating row backgrounds
